# Birhan Energies: Brent Oil Price Analysis

This repository contains a Bayesian Change Point Analysis of Brent oil prices (1987-2022) to understand the impact of geopolitical and economic events.

## Project Status: Interim Submission
- **Task 1 Completed**: Data analysis workflow defined, 12 key events researched, and EDA performed.
- **Documentation**: Interim report available in `docs/interim_report.md`.
- **Data**: Processed price data and structured event CSVs are in the `data/` folder.
- **Visualizations**: Initial price-event correlation plots generated in `notebooks/eda_analysis.ipynb`.

- Python (Pandas, PyMC, Statsmodels, Matplotlib)
